# Proyecto Final - Sistema de Administración de Empleados

Sistema el cual permite la administración de empleados mediante un sistema seguro de autenticación.
La aplicación fue desarrollada utilizando **Node.js** y **Express.js**, implementando autenticación mediante **JWT (JSON Web Token)** para restringir el acceso únicamente a usuarios autorizados.
El sistema permite a los administradores gestionar la información de empleados realizando operaciones CRUD (Crear, Consultar, Actualizar y Eliminar).

# Características principales

- Inicio de sesión seguro para administradores.
- Autenticación mediante JWT.
- Registro de empleados.
- Modificación de información de empleados.
- Eliminación de empleados.
- Búsqueda de empleados por nombre.
- Persistencia de datos en base de datos.
